Set the current tab speed to specified default speed.

You can set more hotkeys by navigating to chrome://extensions/shortcuts.

You can set a default video speed using the above input field and press enter or the 'Set Speed' button. The default speed amount is 1x (normal speed).

Similar to the default speed, you can set an increment amount using the above input field and press enter or the 'Set Increment' button. The default increment amount is 0.25x.

Any video equal to or under the duration entered will not have its speed altered and will remain at 1x speed. The format is MM:SS and can be set by inputing the desired cutoff and pressing the 'Set Cutoff' button or the enter key inside the cutoff input field.
